![windows build](https://github.com/wolodyx/numgeom/actions/workflows/windows.yml/badge.svg)

**Быстрая разработка CAD/CAE/CAM приложений на C++**

NumGeom -- это кроссплатформенный фреймворк, объединяющий передовые технологии 3D-моделирования в единой среде.
Проект предоставляет готовую инфраструктуру для создания интерактивных приложений инженерного анализа, автоматизированного проектирования и подготовки производства.
