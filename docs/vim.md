Для работы с vim установите плагины:
* [vim addon local vimrc](https://github.com/MarcWeber/vim-addon-local-vimrc)
* [editorconfig-vim](https://github.com/editorconfig/editorconfig-vim)

Как установить плагины?
В каталоге `~/.vim/pack/local/start` склонируйте хранилище проекта с плагином из GitHub.
Пример установки `editorconfig-vim`:
``` bash
mkdir -p ~/.vim/pack/local/start
cd ~/.vim/pack/local/start
git clone https://github.com/editorconfig/editorconfig-vim.git
```
